# 🚀 Ninja Cyber JS Toolkit

A modern **JavaScript Toolkit Web App** that provides multiple useful tools in a single interface.
Built using **HTML, CSS, and JavaScript** with a clean UI and responsive layout.

---

## 🔥 Features

- 🧮 Calculator (evaluate expressions)
- 📏 KM to Miles Converter
- 🌡️ Celsius to Fahrenheit Converter
- 🔢 Odd / Even Checker
- 🔍 Positive / Negative Number Checker
- 🎲 Random Number Generator

---

## 📁 Project Structure

```bash
js-toolkit/
│── index.html   # Main UI
│── style.css    # Design & layout
│── script.js    # Logic & functions
```

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ⚙️ How It Works

- User inputs value in fields
- Button click triggers JavaScript functions
- Output is displayed instantly on screen

Example:

- Calculator → uses `eval()`
- Temperature → formula `(C × 9/5) + 32`
- Random → `Math.random()`
